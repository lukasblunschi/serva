Upgrade instructions of Serva
------------------------------------

Usage:
- Always start with the update block of your current version and move up.
  E.g. if you are running version 2.0.1, start with "from 2.0.x" and continue with "from 2.1.x".
- Execute the update instructions within an update block top-down.
  E.g. for update block "2.1.x", rename coach to nurse, then rename group to patient and so on.


from 0.0.0 to 0.1.0:
  - install serva.
