Releasing Serva Howto
---------------------

1. update changelog.txt.

2. update upgrade.txt.

3. increase version number in:
   - src/ch/serva/ServaConstants.java
   - war/WEB-INF/web.xml
   - release.xml
   - run-standalone.sh

4. commit "releasing ..."

5. create git tag

6. run release.xml
   $ ant -f release.xml

7. upload to sourceforge
   - using web interface
