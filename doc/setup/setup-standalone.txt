Setup of Serva as a standalone application
------------------------------------------

1. Create database

   $ sudo mariadb
   > status
     (check that everything is stored in utf8)
   > create database serva;
   > grant all on serva.* to serva@localhost identified by '123';
   > flush privileges;
   > exit

   $ mariadb -u serva --password=123 serva
   > show databases;
   > status
     (check that everything is stored in utf8)
   > exit


2. Restore empty database

   $ cd serva/scripts/
   $ ./restore-db.sh empty.sql


3. Start Serva standalone

   $ cd serva/
   $ ./run-standalone.sh


4. Point your web browser to

   http://localhost:8081/serva


5. Enjoy!
